# 🚀 [채용] Sandbox Network DXD팀 Project Manager (PM)

## 👋 DXD팀을 소개합니다

Sandbox Network의 DXD팀은 **크리에이터 비즈니스를 위한 IT 서비스와 플랫폼을 설계하고 운영하는 CTO 산하 핵심 조직**입니다.

> DXD팀의 Project Manager는 단순한 서비스 기획자가 아니라 **비즈니스와 기술을 연결하는 역할**을 수행합니다.

샌드박스의 방대한 크리에이터 IP와 데이터, 그리고 AI 기술을 활용해 기존 업무 프로세스를 디지털화하고 새로운 플랫폼 서비스를 만들어갑니다. 우리는 Generative AI와 최신 IT 기술을 적극적으로 활용하여 기존 방식보다 훨씬 높은 생산성과 효율을 만드는 것을 목표로 합니다.

**AI 기반 서비스 기획과 플랫폼 구축을 함께 만들어갈 Project Manager를 찾고 있습니다.**

---

## 💻 DXD팀 Project Manager는 이런 일을 합니다

* **AI 기반 서비스 및 플랫폼 기획**
  * 크리에이터 비즈니스를 위한 IT 서비스 기획 및 플랫폼 프로젝트 리딩
  * 서비스 기획 → 개발 → 출시 → 운영까지 End-to-End 프로젝트 관리
* **AI 활용 MVP 검증**
  * LLM 및 AI 도구를 활용하여 기획 아이디어를 MVP 형태로 구현
  * 기술적 타당성 및 서비스 가능성 검증
* **비즈니스 요구사항 시스템 설계**
  * 영업, 재무, 정산 등 다양한 조직의 요구사항을 분석
  * PRD, 기능정의서, 정책 문서 작성
* **사용자 경험 기반 서비스 설계**
  * Figma / FigJam을 활용한 UX 및 서비스 화면 설계
  * 유저 시나리오 기반 서비스 프로세스 설계
* **프로젝트 커뮤니케이션 및 조율**
  * 개발, 사업, 재무, 법무 등 다양한 조직과 협업
  * 프로젝트 일정 및 우선순위 관리
* **협업 환경 및 업무 생산성 개선**
  * SaaS 기반 협업 인프라 운영 (Monday.com, Cloud 등)
  * Generative AI를 활용한 업무 효율화
